## Image Specifications

- **Format**: PNG (preferred) or JPG
- **Aspect Ratio**: 16:9 (recommended)
- **Dimensions**: 1920x1080 or similar high resolution
- **File Size**: Optimize for web (aim for < 500KB per image)
- **Naming**: Use exact filenames listed above (case-sensitive)

## Optimization Tips

Before adding images:
1. Compress images using tools like TinyPNG, ImageOptim, or similar
2. Ensure images are clear and readable at full size
3. Test that images display correctly in the grid layout
4. Verify alt text matches the image content
